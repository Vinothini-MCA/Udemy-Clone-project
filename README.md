# Udemy-Clone-project
This project is a Udemy Clone inspired by the design of Udemy, built using only HTML and CSS. The goal of this project is to practice frontend development and layout design by recreating a popular online learning platform interface.

# Key Features:

Responsive course listing layout
Navigation bar with search functionality design
Structured HTML layout for course sections
Beginner-friendly frontend project for practice

# Technologies Used:

HTML5
CSS3

This project helped me improve my HTML structure, CSS styling, and webpage layout skills while recreating a real-world website interface.
